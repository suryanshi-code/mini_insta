# femagram
A mini social media web app built using Flask.

## 🚀 Features
- User login & signup
- Create posts
- Like posts
- Delete posts

## 🛠️ Tech Stack
- Python (Flask)
- HTML, CSS
- SQLite

## 📸 Future Improvements
- Comments system
- Profile page
- Image upload
